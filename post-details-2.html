C'est quoi le Css

  • Anatole De Vos
  • Date : 8 juin 2023
  • Catégorie : Web

​Le CSS (Cascading Style​​ Sheets) est un​​ langage informat​​ique de description​​ utilisé pour​​ compléter le​​ HTML et lui conf​​érer un style​​ personnalisé

Il permet de créer des pages web à l'apparence soignée en définissant la présentation et la mise en forme des éléments HTML

il est Facile à apprendre et est très Utile.

Il change l'apparence de texte, la taille d'une image, créer des bordures.

.example { 
background-color: #a6e22e;
margin: 10px;
}

Le CSS avec Bootstrap

    Il y a plusieurs types de conteneurs, mais voici les différences entre les deux types de conteneurs mentionnés dans les résultats de recherche :
    • Container: Un container est un élément de virtualisation au niveau de l'application qui permet d'empaqueter et de déployer des applications de manière immuable . Les conteneurs sont plus légers que les machines virtuelles et partagent le même noyau d'exploitation que l'hôte, ce qui les rend plus rapides et plus efficaces
    • Container fluide: Un container fluide (ou "container-fluid" en anglais) est un type de conteneur utilisé dans le système de grilles de Bootstrap pour définir un conteneur de taille fluide, c'est-à-dire un conteneur dont la taille va changer en même temps que celle de la fenêtre de l'utilisateur
    Une row est un élément HTML qui définit une ligne de cellules dans un tableau En résumé, une row est un élément HTML qui définit une ligne de cellules dans un tableau.Html à bootstrap et les classes Css sont ajoutées, pas besoin de les Changer.
    Le système de grille de Bootstrap est un système de disposition principal reposant sur le modèle des boites flexibles (flexbox). Le système de grille de Bootstrap est basé sur un système de 12 colonnes de base. Les colonnes peuvent être combinées pour créer des colonnes plus larges. Le système de grille de Bootstrap est également réactif, ce qui signifie que les colonnes se réorganisent automatiquement en fonction de la taille de l'écran.
